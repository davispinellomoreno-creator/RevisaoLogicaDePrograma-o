 Sobre o Projeto

Este projeto tem como objetivo reforçar e revisar conceitos fundamentais de lógica de programação. A prática constante desses conceitos é essencial para manter uma base sólida, especialmente em um cenário tecnológico dinâmico, onde novas ferramentas e paradigmas surgem com frequência.

A proposta aqui é revisitar problemas clássicos e exercitar o raciocínio lógico, ajudando a relembrar e consolidar conhecimentos que, com o tempo, podem ser esquecidos. Afinal, uma boa lógica é a base para escrever códigos mais eficientes, organizados e fáceis de manter.

Exercícios Resolvidos até o omomento: 

1. Par ou ímpar

Peça um número inteiro ao usuário e informe se ele é par ou ímpar.

2. Maior de três números

Leia três números e exiba qual é o maior deles.

3. Tabuada

Solicite um número e mostre sua tabuada de 1 a 10.

4.Soma de números pares

Leia um número N e calcule a soma de todos os números pares de 1 até N.

5.Número primo

Verifique se um número informado é primo ou não
